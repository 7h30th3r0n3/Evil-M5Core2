Evil-M5Project

What is Evil-M5Project?

Evil-M5Project is an innovative tool developed for ethical testing and exploration of WiFi networks. It's compatible with Cardputer, Atoms3, Fire, Core2. You can scan, monitor, and interact with WiFi networks in a controlled environment. This project is designed for educational purposes, aiding in understanding network security and vulnerabilities.

Get Involved

Evil-M5Project is open-source and available for educational purposes. Follow the GitHub repository to contribute or report issues.

Key Features

  • WiFi Network Scanning
    Identify and display nearby WiFi networks.
  • Network Cloning
    Check information and replicate networks for in-depth analysis.
  • Captive Portal Management
    Create and operate a captive portal to prompt users with a page upon connection.
  • Credential Handling
    Capture and manage portal credentials.
  • Remote Web Server
    Monitor the device remotely via a simple web interface that can provide credentials and upload portals that store files on an SD card.
  • Sniffing Probes
    Sniff and store nearby probes on an SD card.
  • Karma Attack
    Try a simple Karma Attack on a captured probe.
  • Automated Karma Attack
    Try Karma Attack on nearby probes automatically.
  • Bluetooth Serial Control
    Control the device via Bluetooth.
  • Wardriving
    Wardriving with Wigle format output on SD.
  • Wardriving Master
    Use multiple ESP32 devices to monitor all Wi-Fi channels simultaneously for enhanced wardriving capabilities.
  • Beacon Spam
    Generate multiple SSIDs around you.
  • Deauthentication Attacks
    Send deauthentication frames and sniff 4-Way handshakes and PMKID.
  • Handshake Master
    Capture EAPOL frames across multiple Wi-Fi channels using Sniffer Slaves.
  • Client Sniff And Deauth
    Sniff clients connected to AP and auto deauth while sniffing EAPOL.
  • EAPOL/Deauth Detection
    Detect deauthentication packets, 4-Way handshakes, PMKID, and pwnagotchi near you.
  • Wall Of Flipper
    Detect and save Flipper Zero with Bluetooth enabled near you and detect BLE SPAM.
  • Send Tesla Code with RFunit
    Use RFunit to send Tesla codes, mimicking Flipper Zero capabilities.
  • Scan Network and Port
    Perform network and port scans to discover devices and services, checking hosts' status.
  • SSH Shell
    Connect to SSH servers directly from the device, allowing command execution via an on-device shell.
  • Web Crawler
    Crawl websites to extract information, ensuring authorization to crawl before use.
  • PwnGrid
    Spam face and message on pwnagotchi devices nearby, causing a Denial of Screen PWND.
  • Skimmer Detector
    Detect potential Bluetooth skimmers using HC-03, HC-05, and HC-06 modules.
  • BadUSB
    Simulate BadUSB attacks by emulating keyboard/mouse inputs to execute predefined scripts or commands.
  • Full Network Scan
    Complete network discovery, identify all connected devices, open ports, and services. Store and retrieve scan data for future reference.
  • Web Scraping for Full Web Presence Analysis
    Scrape detected web pages across the network for further investigation.
  • Custom Theming
    Customize EvilCardputer's interface colors via theme.ini in your SD card's root.
  • Wi-Fi Channel Visualizer
    Visualize the usage of Wi-Fi channels in the 2.4GHz band.
